# CIP 2020 Four-Digit Taxonomy

## Source
Classification of Instructional Programs (CIP) 2020 edition, published by the
National Center for Education Statistics (NCES), U.S. Department of Education.

## Purpose
This file contains the complete set of four-digit CIP 2020 codes compiled from
the NCES website. Each entry includes the CIP code, official program name,
two-digit domain prefix, domain name, and a set of short aliases — abbreviations,
nicknames, and catalog naming variations that a college student would realistically
type when searching for a course or discipline.

## Schema
| Field        | Type     | Description                                         |
|--------------|----------|-----------------------------------------------------|
| `code`       | string   | Four-digit CIP code (e.g. `"14.07"`)                |
| `name`       | string   | Official NCES program name                          |
| `domain`     | string   | Two-digit series prefix                             |
| `domainName` | string   | Series-level title from NCES                        |
| `aliases`    | string[] | 2-5 lowercase search aliases for student matching   |
