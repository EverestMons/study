Specify: where the rating data comes from (PracticeMode already stores it), the calibration formula, the UI components, and the data flow. **Also scope:** (B2) **Review mode entry point (IES Rec 1, optional):** Assess whether adding a 6th study mode ("Review") to the existing 5-mode picker is worth it given that due skills are already surfaced via badges. If no, document why and close the gap as "addressed by existing DUE badges." **Blueprint must include** a "How to verify" section with per-item acceptance criteria. **Deposit:** `study/knowledge/architecture/ies-gaps-open-flags-blueprint-2026-03-22.md`.
